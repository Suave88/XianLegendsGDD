üß™ Testing Phases

Phase 1-2: Internal Testing

Focus: Core systems, combat mechanics, plugin stability
Who: Dev team only
Goal: Catch critical bugs early, validate mechanics work

Phase 3-4: Alpha Testing

Focus: Gameplay balance, story flow, Base Camp features
Who: Dev team + 2-3 trusted friends
Goal: First impressions, pacing feedback, major bugs

Phase 5: Beta Testing

Focus: Full playthrough, all systems, final polish
Who: 5-10 external playtesters
Goal: Comprehensive bug hunting, balance refinement, UX feedback

Phase 6: Release Candidate

Focus: Critical bugs only, stability testing
Who: Dev team final pass
Goal: Ensure launch-ready build with no game-breaking issues
